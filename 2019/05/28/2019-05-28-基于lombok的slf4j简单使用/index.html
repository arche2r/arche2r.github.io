基于lombok的slf4j简单使用

lombok is a java library that automatically plugs into your editor and build tools, spicing up your java.
Never write another getter or equals method again, with one annotation your class has a fully featured builder, Automate your logging variables, and much more.

今天主要谈简单使用lombok注解+slf4j+slf4j-simple来快速在控制台打印日志.

SLF4J，即简单日志门面（Simple Logging Facade for Java），不是具体的日志解决方案，它只服务于各种各样的日志系统

阿里爸爸的开发手册中强制使用slf4j来作为日志系统的框架,可以极为方便的在以后切换不同日志系统.而且slf4j的占位符系统和性能优化也是bug级别的存在.

首先在ide中添加lombok插件,然后添加lombok和slf4j-simple的依赖包

        <!--注意jar包为slf4j-simple-->
        <dependency>
            <groupId>org.slf4j</groupId>
            <artifactId>slf4j-simple</artifactId>
            <version>1.8.0-alpha2</version>
        </dependency>
        <!--支持lombok的jar包-->
        <dependency>
            <groupId>org.projectlombok</groupId>
            <artifactId>lombok</artifactId>
            <version>1.16.20</version>
        </dependency>

然后编写测试类

//基于lombok的注解,可以省去log的定义
@Slf4j
public class Demo {
    public static void main(String[] args) {

        log.info("这里是{}的日志信息","Archer");
    }
}
